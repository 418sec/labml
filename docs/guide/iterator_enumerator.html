Iterators and EnumeratorsÂ¶

You can use iterate() and enum() with any iterable object. In this example we use a PyTorch DataLoader.

# Create a data loader for illustration
import torch
from torchvision import datasets, transforms

test_loader = torch.utils.data.DataLoader(
        datasets.MNIST('./data',
                       train=False,
                       download=True,
                       transform=transforms.Compose([
                           transforms.ToTensor(),
                           transforms.Normalize((0.1307,), (0.3081,))
                       ])),
        batch_size=32, shuffle=True)

for data, target in logger.iterate("Test", test_loader):
    time.sleep(0.01)

Test...[DONE] 6,618.08ms

for i, (data, target) in logger.enum("Test", test_loader):
    time.sleep(0.01)

Test...[DONE] 6,662.78ms
